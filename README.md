# leave-gantt · 請假時程甘特圖規劃器

互動式單檔網頁，依公司請假辦法與年度行事曆，試算「懷孕 → 育嬰留停」的完整請假時程。

接力順序：**婚假 → 年假 → 安胎假 → 產假 → 育嬰留停**，目標：越早離開職場。

## 功能
- 受孕日、預產期、產假開始、年假/婚假/產檢假天數、育嬰留停月數皆可自由調整（不寫死）
- 安胎假開始可用滑桿或拖拉甘特圖圓點即時調整
- 國定假日清單可編輯（影響工作日計算），預載年度行事曆值 + 後續年度推估
- 即時甘特圖、薪資（全薪/無薪/就保津貼）統計、時程明細表

> 本工具為試算，非個案法律意見；實際請以人資與主管核定為準。
